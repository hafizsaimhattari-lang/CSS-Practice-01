# Basic CSS Properties Overview

## Basic CSS Properties kya hoti hain?

CSS properties woh instructions hoti hain jo browser ko batati hain ke HTML element kaisa dikhna chahiye.

Example:

```css
p {
  color: red;
}
```

Yahan `color` property hai aur `red` uski value hai.

## `color` property

`color` property text ka color change karti hai.

Example:

```css
h1 {
  color: blue;
}
```

## `background-color` property

`background-color` property kisi element ka background color change karti hai.

Example:

```css
div {
  background-color: black;
}
```

## Dono properties me farq

| Property | Kaam |
| --- | --- |
| `color` | Text ka color change karti hai |
| `background-color` | Element ke background ka color change karti hai |

## Practice ka tareeqa

HTML me elements banao.

CSS me un elements ko selector se target karo.

Phir property aur value likho.

Example:

```css
p {
  color: white;
  background-color: black;
}
```

Is example me paragraph ka text white aur background black ho jayega.
