Text Align Practice

Initial Value

HTML and CSS are the building blocks of every website. Understanding how text alignment works helps developers create clean and professional layouts. Experiment with different values and carefully observe how the text changes on the page.

Unset Value

CSS provides several keyword values such as inherit, initial, and unset. Although they may look similar, each one behaves differently depending on the property being used. Practice them yourself instead of only reading about them.

Compare Both

Compare the output of the first and second articles. Apply different text-align values to the parent elements and then use initial or unset on the paragraphs. Carefully notice which paragraph follows its parent and which one returns to the browser's default value.
