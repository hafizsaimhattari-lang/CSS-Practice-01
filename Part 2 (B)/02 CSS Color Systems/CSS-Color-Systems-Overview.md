# CSS Color Systems Overview

## CSS Color Systems kya hotay hain?

CSS me colors likhne ke multiple tareeqay hotay hain.

Part 2 ke is section me hum basic color systems practice kar rahe hain.


## Named Colors

Named colors woh colors hotay hain jo naam se likhe jate hain.

Example:

```css
h1 {
  background-color: red;
}
```

Yahan `red` named color hai.

## RGB Colors

RGB ka full form hai:

```text
Red Green Blue
```

RGB me har value `0` se `255` tak hoti hai.

Example:

```css
body {
  background-color: rgb(2, 67, 67);
}
```

## Hexadecimal Colors

Hex color `#` se start hota hai.

Example:

```css
h1 {
  color: #ffffff;
}
```

Yahan `#ffffff` white color hai.

## Short comparison

| Color System | Example | Use |
| --- | --- | --- |
| Named color | `red` | Easy practice |
| RGB | `rgb(255, 0, 0)` | Exact red/green/blue control |
| Hex | `#ff0000` | Common web design format |

## Yaad rakhne wali baat

Har color system ka kaam same hai: color define karna.

Farq sirf likhne ke tareeqay ka hota hai.
