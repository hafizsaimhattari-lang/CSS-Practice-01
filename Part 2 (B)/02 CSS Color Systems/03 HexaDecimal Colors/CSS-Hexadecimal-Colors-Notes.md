# CSS Hexadecimal Colors

## Hexadecimal color kya hota hai?

Hexadecimal color CSS me color likhne ka common web format hai.

Hex color `#` se start hota hai.

Example:

```css
h1 {
  color: #ffffff;
}
```

## Basic syntax

```css
selector {
  color: #rrggbb;
}
```

Yahan:

- `rr` red value
- `gg` green value
- `bb` blue value

## Is practice me kya ho raha hai?

CSS me body background hex color se set hai:

```css
body {
  background-color: #044040;
}
```

Heading ka text white hai:

```css
h1 {
  color: #ffffff;
}
```

## Common hex colors

| Hex | Color |
| --- | --- |
| `#ffffff` | white |
| `#000000` | black |
| `#ff0000` | red |
| `#00ff00` | green |
| `#0000ff` | blue |

## Hex aur RGB ka relation

Ye dono same color represent kar sakte hain.

Example:

```css
color: #ff0000;
color: rgb(255, 0, 0);
```

Dono red color hain.

## Common mistakes

- `#` symbol miss karna.
- Hex value me invalid letters use karna.
- 6 digits ki jagah incomplete value likhna.
- Text/background contrast ka khayal na rakhna.

## Short summary

Hex colors web design me bohat common hain.

Ye compact format me exact colors define karne ke liye use hotay hain.
