# CSS Named Colors

## Named colors kya hotay hain?

Named colors CSS me colors likhne ka sab se simple tareeqa hai.

Is me hum color ka naam directly likhte hain.

Example:

```css
h1 {
  color: red;
}
```

## Is practice me kya ho raha hai?

HTML me 3 headings hain:

```html
<h1 class="box-1">Red Color Ka Dabba</h1>
<h1 class="box-2">Blue Color Ka Dabba</h1>
<h1 class="box-3">Gold Color Ka Dabba</h1>
```

CSS me in classes ko named colors diye gaye hain:

```css
.box-1 {
  background-color: red;
}

.box-2 {
  background-color: blue;
}

.box-3 {
  background-color: gold;
}
```

## Important point

Named colors easy hote hain, lekin har exact shade ka name available nahi hota.

Is liye professional design me RGB, Hex, HSL bhi use hotay hain.

## Common mistakes

- Color name ki spelling galat likhna.
- Text aur background dono same color rakh dena.
- `background-color` ki jagah `background color` likhna.

## Short summary

Named colors beginner practice ke liye best hain.

Examples:

- `red`
- `blue`
- `gold`
- `white`
- `black`
