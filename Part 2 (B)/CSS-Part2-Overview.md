## Basic CSS Property Rule

```css
h1 {
  color: white;
  background-color: black;
  text-align: center;
}
```

Yahan:

- `h1` selector hai
- `color`, `background-color`, aur `text-align` properties hain
- `white`, `black`, aur `center` values hain

## Yaad rakhne wali baat

Selector element ko target karta hai.

Property us element ka design change karti hai.

Value batati hai ke design kaisa hona chahiye.
