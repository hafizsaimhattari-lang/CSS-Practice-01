## Is topic ka main idea

`:focus` tab apply hoti hai jab user kisi element ko select kare.

Example:

```css
input:focus {
  border: 3px solid red;
}
```

Iska matlab:

Jab input field focus me aaye, uska border red ho jaye.

## Yaad rakhne wali line

`:focus` selected input, button, link, ya form element ko highlight karne ke liye use hoti hai.
