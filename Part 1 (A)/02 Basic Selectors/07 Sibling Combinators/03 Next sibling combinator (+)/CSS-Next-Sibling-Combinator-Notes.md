# 🎯 03 Next Sibling Combinator (`+`)

## 📖 Kya Hai?
`+` combinator kisi element ke **turant agle ek sibling** ko select karta hai.

```css
h1 + p { color: green; }   /* h1 ke bilkul baad wala pehla p */
```
