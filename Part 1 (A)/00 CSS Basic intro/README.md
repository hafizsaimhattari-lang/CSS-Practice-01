## 📖 CSS Kya Hai?
**CSS (Cascading Style Sheets)** ek style language hai jo HTML elements ko dekhne mein khubsoorat banati hai.

- HTML = Structure (Dhancha)
- CSS = Design (Sajawat)
- JavaScript = Behaviour (Harkat)

---

## 🔑 Key Concepts

| Concept         | Matlab                                      |
|-----------------|---------------------------------------------|
| Selector        | Konsa element style karna hai               |
| Property        | Kya style karna hai (color, font-size, etc) |
| Value           | Style ki value (red, 16px, bold, etc)       |

### Basic Syntax:
```css
selector {
    property: value;
}
```

### Example:
```css
h1 {
    color: blue;
    font-size: 32px;
}
```
