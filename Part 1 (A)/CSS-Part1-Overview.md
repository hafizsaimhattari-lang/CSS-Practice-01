## 🎯 Folder Details

### 📁 00 CSS Basic intro
CSS kya hoti hai, browser ise kaise parh ta hai, aur HTML se kaise jurti hai — yeh folder yahan se shuru hota hai.

### 📁 01 Types of CSS
CSS likhne ke 3 tareeqay hote hain:
- **Inline CSS** → Style seedha HTML tag ke andar `style=""` se.
- **Internal CSS** → `<style>` tag HTML ke `<head>` mein.
- **External CSS** → Ek alag `.css` file banao aur `<link>` se connect karo ✅ (Professional Tareeqa).

### 📁 02 Basic Selectors
CSS mein kisi bhi element ko select karne ke liye selectors use hote hain:
- **Element Selector** → `p { }`, `h1 { }`
- **Class Selector** → `.myClass { }`
- **ID Selector** → `#myId { }`
- **Universal Selector** → `* { }`
- **Grouping Selector** → `h1, h2, p { }`
- **Descendant Selector** → `div p { }`
- **Sibling Combinators** → `+` aur `~` se
