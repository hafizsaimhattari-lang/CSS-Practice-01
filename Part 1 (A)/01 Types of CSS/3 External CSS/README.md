## 📖 External CSS Kya Hai?
Ek alag `.css` extension wali file banai jati hai jisme saari CSS likhi jati hai. Phir HTML file mein `<link>` tag se wo file connect ki jati hai.

**HTML File:**
```html
<head>
    <link rel="stylesheet" href="style.css">
</head>
```

**CSS File (style.css):**
```css
h1 {
    color: red;
    font-size: 36px;
}
p {
    color: gray;
}
```
