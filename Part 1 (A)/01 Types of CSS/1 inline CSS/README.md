## 📖 Inline CSS Kya Hai?
Inline CSS mein `style=""` attribute seedha HTML tag ke andar likha jata hai.

```html
<h1 style="color: blue; text-align: center;">Heading</h1>
<p style="color: green; font-size: 18px;">Paragraph</p>
```
